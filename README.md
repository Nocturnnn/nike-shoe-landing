# Nike 3D Product Experience

Experiência visual focada em showcase de produto utilizando WebGL.

## Preview
(video ou gif)

## Stack
- React
- TypeScript
- Three.js
- React Three Fiber
- Tailwind CSS
- GSAP
- Framer Motion

## Objetivo
Explorar motion design, interação com scroll e visualização 3D para páginas de produto premium.

## Como rodar o projeto

1. Clone o repositório

git clone https://github.com/seuusuario/nome-do-projeto.git

2. Entre na pasta do projeto

cd nome-do-projeto

3. Instale as dependências

npm install

4. Inicie o servidor de desenvolvimento

npm run dev

5. Abra com o link fornecido ex: http://localhost:5173/
